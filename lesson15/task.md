## Класс для представления графа с помощью матрицы смежности

Необходимо реализовать класс для описания графа на основе типизированного массива (Uint8Array, Uint16Array и других).

Граф может быть ориентированным и неориентированным.
В ячейках матрицы можно хранить вес ребра или дуги (0 означает отсутствие связи). Весом может быть как число, так и другой View.

```js
// Пример использования
const matrix = new Matrix(Uint8Array, 10, 10);  // 10×10, значения 0-255

// Заполняем матрицу смежности
// ...

const graph = new Graph(matrix);

// Проверяем смежность двух узлов (с учётом направленности)
graph.hasEdge(1, 2);      // Для неориентированного графа
graph.hasArc(1, 2);       // Для ориентированного графа (направление важно)

// Добавляем ребро между двумя узлами (с опциональным весом)
graph.addEdge(7, 2, weight);

// Удаляем ребро между двумя узлами
graph.removeEdge(7, 2);

// Добавляем дугу (для ориентированного графа)
graph.addArc(7, 2, weight);

// Удаляем дугу
graph.removeArc(7, 2);
```

## Метод traverse для обхода смежных узлов графа начиная с заданного

Необходимо реализовать метод traverse для обхода графа начиная с заданного узла.

Можно использовать обход в глубину (DFS) или в ширину (BFS). Уже пройденные узлы нужно игнорировать (чтобы избежать зацикливания).

```js
const matrix = new Matrix(Uint8Array, 10, 10);

// Заполняем матрицу смежности
// ...

const graph = new Graph(matrix);

graph.traverse(1, (node, depth) => {
console.log(`Узел: ${node.id}, глубина: ${depth}, вес ребра: ${node.weight}`);
});
```
