# Замеры производительности PIXEL STREAM

## ОБЩИЙ РЕЙТИНГ (по скорости операций)

| Место | Реализация                    | Сильные стороны                      | Слабые стороны                    |
|-------|-------------------------------|--------------------------------------|-----------------------------------|
| 🥇    | **Uint8ArrayPixelStream**     | Самый быстрый в большинстве операций | ColMajor обход медленнее RowMajor |
| 🥈    | **FlatArrayPixelStream**      | Отличная скорость записи             | Медленный ColMajor обход          |
| 🥉    | **ArrayOfObjectsPixelStream** | Хорош для ColMajor                   | Медленная запись                  |
| 4     | **ArrayOfArraysPixelStream**  | Быстрый forEach RowMajor             | Очень медленная запись            |

## Увеличение яркости. Обход + модификация (1M пикселей)

| Реализация         | Циклы (ms) | forEach (ms) | Победитель |
|--------------------|------------|--------------|------------|
| **Uint8Array**     | 14.86      | 16.86        | ✅ Циклы    |
| **FlatArray**      | 14.93      | 17.25        | ✅ Циклы    |
| **ArrayOfObjects** | 19.37      | 21.09        | ✅ Циклы    |
| **ArrayOfArrays**  | 41.56      | 33.43        | ✅ forEach  |

Было два бенчмарка (обход в цикле и использование реализации forEach).
**Вывод:** Для ArrayOfArrays forEach быстрее на 20%, для остальных циклы быстрее на 13-16%. При этом весь обход занимает больше времени в сравнении с другими реализациями.

## Рекомендации

### Выберите **Uint8ArrayPixelStream** если:
- Нужна сбалансированная производительность
- Важна экономия памяти (4 байта на пиксель)
- Смешанная нагрузка (чтение + запись)
- Cache locality критична

### Выберите **FlatArrayPixelStream** если:
- Много операций записи
- Нужна максимальная скорость setPixel
- Работаете с большими объемами (4M+)

### Выберите **ArrayOfObjectsPixelStream** если:
- ColMajor обход эффективней RowMajor
- Удобство работы с объектами приоритетнее скорости

### Выберите **ArrayOfArraysPixelStream** если:
- Только чтение (forEach RowMajor)
- Редкая запись
- ⚠️ Избегайте массовой записи (самый медленный)
